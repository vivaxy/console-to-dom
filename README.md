# Usage

add this to debug html

```
<script type="text/javascript" src="../dest/index.js"></script>
<script>
    consoleToDom();
</script>
```

# [Demo](http://vivaxy.github.io/console-to-dom/demo)

# Develop

1. `> npm i`

2. `> npm run build`

with webpack-dev-server `> npm run server`
